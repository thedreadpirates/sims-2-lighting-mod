adjusting shadow darkness
is what affects the overall shadow darkness and ambiant light. More so for dark areas or unlit rooms. threshold is how much light is needed before ambiant light takes over

to edit open the lighting.txt file inside the sims 2 base game. 

insert here

if you are using deluxe its in

insert here

then when its open serch for summer
and below that find unlitroomcolor

you will then see after it something like (.45, .425, .4) -threshold .4

which will mean the following:

(Red, Green, Blue) -threshold float

use the following below for a more quicker change of brightness. copy

for others search for:
Dawn
Dusk
Nighttime

change the unlitroomcolor to your liking

If you have Seasons installed you will also have to edit WINTER, SPRING and AUTUMN. They will use the Day settings as well.

NOTE: that I've mainly use the RLS unlitroomcolor settings and not any of the others as I recommand using the RLS default over everything else.


# Controls non lit objects, rooms, sims total ambiant light.
# Formula:
# (R,G,B) -threshold float
# Threshold determines how much introduced light there must be
# before ambiant light takes over.

---------
--Maxis--
---------
	-Day-
   unlitRoomColor (.45, .425, .4) -threshold .4

	-Night-
   unlitRoomColor (0.345, 0.345, 0.42) -threshold .4

	-Dusk and Dawn-
   unlitRoomColor (.45, .425, .4) -threshold .4

-----------------------------------------------
--dDefinder Brighter (Not as bright as Maxis)--
-----------------------------------------------
	-ALL-
   unlitRoomColor (.2, .2, .25) -threshold .4
	
---------------------------------------------
--Chocololate Pi (Brighter Night, darker day)--
---------------------------------------------
	-Day-
   unlitRoomColor (0.03, 0.03, 0.035) -threshold .5

	-Night-
   unlitRoomColor (0.1, 0.1, 0.1) -threshold .5

	-Dawn-
   unlitRoomColor (0.03, 0.03, 0.035) -threshold .5

	-Dusk-
   unlitRoomColor (0.1, 0.1, 0.1) -threshold .9

----------------------------------------
--dDefinder older versions (Darker)--
----------------------------------------
	-Day-
   unlitRoomColor (0.03, 0.03, 0.035) -threshold .5

	-Night-
   unlitRoomColor (0.01, 0.01, 0.01) -threshold .5

--------------------
--Gunmod (Current)--
--------------------
	-ALL-
   unlitRoomColor (.09, .09, .11) -threshold .4
